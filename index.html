Quem sou eu?

Minha história na programação

Sou poetisa, conheci a área de tecnologia através do Gustavo Guanabara e Apaixonei-me pelo Front-end uns 4 meses atrás, pela grande admiração que tinha pelos profissionais que trabalham com ela e pelo iniciativa de um parente meu que me deu um incentivo a estudar sobre a criação de sites, fiz um curso de python no ano passado, abandonei no começo por conta do Enem, já nesse ano, fui direto para HTML e CSS e dai que parti para algoritmos e pensamento computacional, fiz vários projetos (alguns estão no Github). Atualmente estou me aprofundando mais em JavaScript e MySQL. Também sou apaixonada por ciências, literatura brasileira, filosofia, história, música, tudo aquilo que a arte me permite ser e claro... programação. Sei que é um caminho árduo e cheio de dificuldades imensas mais para aqueles que acreditam que a Internet pode mudar vidas e para aqueles que estão aqui nessa mesma jornada, assim como eu, te digo: "Ser programador é ser paciente e persistente na busca de novos conhecimentos." — Igor Barros

Projetos

Projeto 1

Validando formulários com JavaScript

Esse projeto foi feito para desenvolver melhor minhas habilidades de escrita e no JavaScript. Foi uma validação de formulários onde pode colocar os dados e se falhar foi por falta de algum dado.

Projeto 2

Flex turismo

Projeto feito para aprender os conceitos de posicionamento do flexbox. Foi feito em parceria com a DIO no direcionamento do Curso de desenvolvimento das habilidades ali adquiridas, foi um projeto muito importante que ajudou na minha evolução com as caixas no CSS.

Projeto 3

Marketing Criativo

Landing page simples, com a ideia de pegar os conceitos mais simples de uma página. Esse projeto foi um aprendizado a mais nos meus estudos, feito para servir de exemplo para as outras que já fiz e ainda tento melhorar ela.

Projeto 4

Projeto Android

Um dos mais importantes que já fiz em desenvolvimento com o Professor Gustavo Guanabara, agradeço imensamente por tudo que ele me ensinou. Essa é uma página Educacional sobre o Macote da Android que criamos para aplicar todos os conceitos de HTML5 e CSS3

Projeto 5

Tudo sobre o google glass

Projeto feito Pelo cursoVideo para entendemos todos os conceitos de HTML5, o curso foi um pouco antigo, mais foi muito importante para me orientar o que eu queria seguir na carreira. É um site informativo sobre o googleglass onde tem tudo sobre ele.

Projeto 6

Projeto poesia

Esse projeto foi feito com a conclusão de ser retirado o máximo de conhecimento sobre a propiedade de imagens de fundo. O projeto iicial era um cordel, como eu já criava poesia resolvi modificar para um projeto meu
